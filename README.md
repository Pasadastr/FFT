# FFT

По идее поступил не очень хорошо --- не эксплуатировал то, что у нас только ограничееное число простых чиселок в разложении длины, ибо заранее не ведал, как такое делать, поэтому пошёл по пути с курса вычматов, который работает для всех длин тоже за nlogn(но на втором этапе увеличиваем число элементов матрицы в 4 раза...):

 - Так как $jk = \frac{(j+k)^2 - j^2 - k^2}{2}$, то умеем разложить матрицу Фурье в $F = D\cdot T \cdot D'$, где T --- тёплицева
 - Тёплицеву можем <<вложить>> в циркулянтную
 - Циркулянтные как раз и умеем быстро умножать на вектор
